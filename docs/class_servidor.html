Descrição detalhada

Classe que armazena e gerencia as informações de um servidor.

A classe Servidor armazena e gerencia as informações básicas de um servidor, incluindo o ID do dono, nome, descrição, código de convite, lista de canais contidos no servidor e lista de participantes.
